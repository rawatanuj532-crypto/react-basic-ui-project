# React Basic UI Project

This is a beginner-level React project focused on understanding and implementing reusable UI components such as a Navbar, Cards, and a Footer. The project emphasizes clean structure, component-based design, and basic styling.

---

## Features

* Reusable and modular React components
* Multiple Card components rendered using props
* Simple and structured UI layout
* Basic CSS styling for layout and presentation

---

## Project Structure

```
src/
 ├── components/
 │    ├── Navbar.jsx
 │    ├── Footer.jsx
 │    ├── Card.jsx
 │    ├── Card.css
 │
 ├── App.jsx
 ├── main.jsx
```

---

## Tech Stack

* React.js
* JavaScript (ES6)
* CSS

---

## Learning Outcomes

* Building and organizing React components
* Passing and using data through props
* Understanding component reusability
* Structuring a React project effectively

---

## Run Locally

```
npm install
npm run dev
```
